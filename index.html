Setup Instructions

Follow these steps to connect this form to your Google Sheet:

Important: This form must be hosted on a web server (like GitHub Pages) to work with FormSubmit. If you're viewing this as a local file, use the "Export Data" feature instead.

Step-by-Step Guide

  1. Go to sheets.google.com and create a new spreadsheet
  2. Name the first row headers: Timestamp, RollNumber, Name, Subject, Date
  3. Click the "Share" button and set to "Anyone with the link can edit"
  4. Copy your spreadsheet URL and paste it in the form below
  5. Click "Save Configuration" to connect your sheet

Setup Complete!

Your attendance system is now connected to Google Sheets.

Switch to the "Mark Attendance" tab to test the system.
